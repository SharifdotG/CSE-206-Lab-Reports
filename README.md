# CSE 206 - Data Structures Lab

This repo is for the CSE 206: Data Structures Lab course at the Department of Computer Science and Engineering, University of Asia Pacific. This repo specifically contains the Assignments & Lab Reports that was assigned throughout the course. There are a total of 2 Assignments, 1 Lab Report and also contains solution for the Lab Final exam. Obviously the Everything is written in both C and C++ programming language.

## Course Description

This course is designed to provide students with hands-on experience in implementing and using fundamental data structures. The course will cover the implementation and use of arrays, linked lists, stacks and queues. The course will be taught using a high-level programming language such as C or C++.

- **Course Title**: Data Structures Lab
- **Course Code**: CSE 206
- **Semester**: 2-1
- **Credit Hours**: 1.5
- **Prerequisite**: none
- **Faculty**: [Sumaiya Akhtar Mitu](https://cse.uap-bd.edu/faculty/faculty_details/54)
